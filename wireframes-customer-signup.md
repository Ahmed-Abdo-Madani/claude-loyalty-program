## DESIGN SPECIFICATIONS:

### Mobile-First Design:
- Large touch targets (44px minimum)
- Easy thumb navigation
- Clear typography (18px+ body text)
- Minimal scrolling required

### Form UX:
- Auto-focus on first field
- Smart keyboard types (number pad for phone)
- Real-time validation feedback
- Progress indication
- One-tap wallet integration

### Key Features:
- QR code context preservation
- Instant gratification (wallet card)
- Clear value proposition
- Minimal friction signup
- Trust indicators (security, privacy)
